Python Cheat Sheet for Linux

1. Create a Virtual Environment

python3 -m venv my_environment

Creates a new virtual environment named my_environment in the current directory.

2. Activate the Virtual Environment

source my_environment/bin/activate

This command activates the virtual environment. You should see the environment name in your terminal prompt.

3. Install Packages

pip install package_name

Installs the specified package (e.g., Flask, Django) within the activated environment.

4. Check Installed Packages

pip list

Displays a list of all packages installed in the current virtual environment.

5. Write a Simple Python Script

print("Hello, World!")

This script outputs "Hello, World!" to the console when executed.

6. Run the Script

python script_name.py

Executes the Python script specified by script_name.py. Ensure you are in the correct directory.

7. Create an HTML/CSS Interface


<!DOCTYPE html>
<html>
<head></head>
<body>
    <h1>My Python Script</h1>
    <script src="script.js"></script>
</body>
</html>

This basic HTML structure allows you to create a web interface for your Python script.

8. Link Python with HTML


from flask import Flask, render_template

app = Flask(__name__)

@app.route("/")
def home():
    return render_template('index.html')

if __name__ == "__main__":
    app.run(debug=True)

This code creates a Flask application that serves the HTML page index.html.

9. Run the Flask Application

python app.py

This command starts the Flask web server. Ensure your app is named app.py.

10. Access Your Web Application

http://localhost:5000

Open this URL in your web browser to access your Flask application.

11. Handle Form Data in Flask


from flask import Flask, request

@app.route("/submit", methods=["POST"])
def submit():
    name = request.form.get("name")
    return f"Hello, {name}!"

This snippet shows how to handle form submissions in Flask and greet the user.

12. Create a Basic HTML Form


<form action="/submit" method="post">
    <input type="text" name="name" placeholder="Enter your name">
    <input type="submit" value="Submit">
</form>

This HTML code creates a form to collect user input and submit it to the Flask server.

13. Install Flask

pip install Flask

Installs the Flask web framework, enabling you to create web applications in Python.

14. Use Virtual Environments

deactivate

Exits the virtual environment and returns to the system's Python environment.

15. Requirements File

pip freeze > requirements.txt

Creates a requirements.txt file listing all installed packages in the current environment.

16. Install from Requirements File

pip install -r requirements.txt

Installs all packages listed in the requirements.txt file.

17. Debugging Flask Applications

app.run(debug=True)

Enables debug mode in Flask, allowing for easier troubleshooting during development.

18. Handle Static Files


@app.route("/static/")
def static_files(filename):
    return send_from_directory('static', filename)

This function allows you to serve static files like CSS and JavaScript in Flask.

19. Create a CSS File


<link rel="stylesheet" href="{{ url_for('static', filename='style.css') }}">

This line in your HTML links to a CSS file located in the static folder.

20. Deploying Flask Applications


# Using gunicorn
gunicorn -w 4 app:app

This command runs your Flask application using Gunicorn with 4 worker processes for production.
